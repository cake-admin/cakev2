# Spacing, Radius & Stroke

The cake& spacing tokens, generated from the Figma **Value** variable collection
(`& spacing/Value.tokens.json` → `npm run build:cakeand-tokens` →
`npm run build:cakeand-vars`). Each token is a CSS custom property whose name
mirrors the Figma variable 1:1 — `Spacing/Small/space-100` ⇄
`var(--space-100)`. Use these for every padding, gap, margin, corner, and
border width — never arbitrary pixel values.

export const spacing = tokens.value.spacing;
export const radius = tokens.value.radius;
export const stroke = tokens.value.stroke;
export const MUTED = { fontSize: 12, opacity: 0.6 };

## Spacing — `--space-*`

Grouped in Figma as Small (2–10), Medium (12–24), and Large (32–80).

## Radius — `--radius-*`

`--radius-1000` (999px) is the pill/circle radius used by Button, IconButton,
and Switch.

<div style={{ display: 'flex', gap: 24, flexWrap: 'wrap', marginTop: 8 }}>
  {radius.map((t) => (
    <div key={t.name} style={{ textAlign: 'center' }}>
      <div
        style={{
          width: 72,
          height: 72,
          background: '#bcc3ff',
          border: '2px solid #394edb',
          borderRadius: t.px,
        }}
      />
      <div style={{ fontSize: 12, marginTop: 6 }}>
        <code>--{t.name}</code>
        <div style={{ opacity: 0.6 }}>{t.px}px</div>
      </div>
    </div>
  ))}
</div>

outlines `--stroke-100`, focus rings `--stroke-300`).
